- user_skills: skills cargadas para usuarios demo
- job_posts: 3 vacantes
- cv_documents: 3 CVs
- cv_ai_summaries: 3 resúmenes IA de CV
- applications: 3 postulaciones
- application_status_history: historial de cambios
- ai_evaluations: 3 evaluaciones IA
- user_job_recommendations: 3 recomendaciones de vacantes
- company_candidate_requests: 3 solicitudes empresa -> candidato
- notifications: 3 notificaciones demo
- ai_conversations / ai_messages: chat demo usuario/empresa
- admin_ai_settings: settings demo de IA
- admin_audit_logs: log demo de auditoría

IMPORTANTE:
- La UI nueva no consulta todavía esas tablas para la mayoría de pantallas.
- El registro sí fue alineado al esquema simplificado para evitar errores SQL por columnas eliminadas.
